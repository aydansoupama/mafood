# MaFood

MaFood est une application de gestion de nutrition et de planification de repas conçue pour aider les étudiants à mieux gérer leur alimentation.

## Fonctionnalités principales

- Base de données de recettes avec informations nutritionnelles
- Tracker d'ingrédients et de calories
- Enregistrement des recettes préférées
- Planification des repas hebdomadaire
- Intégration avec des outils de santé et de bien-être
